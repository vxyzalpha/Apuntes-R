¿Por qué visualizar datos?

La visualización es una de las habilidades más poderosas en el análisis de datos. Permite detectar patrones que los números solos no revelan, comunicar resultados complejos de forma clara y tomar decisiones basadas en evidencia visual.

🔍

Exploración

Ver distribuciones, outliers y relaciones antes de modelar. El Cuarteto de Anscombe es el ejemplo clásico.

📢

Comunicación

Transformar resultados técnicos en historias visuales comprensibles para cualquier audiencia.

🐛

Diagnóstico

Detectar errores en datos, supuestos de modelos rotos, o comportamientos inesperados.

⚡

Toma de decisiones

Los dashboards y visualizaciones interactivas impulsan decisiones basadas en datos en tiempo real.

ggplot2
La gramática de gráficos

ggplot2 está basado en la Grammar of Graphics de Wilkinson. Todo gráfico se construye combinando capas independientes.

● Nivel: Inicial

La estructura básica

Todo gráfico en ggplot2 sigue esta anatomía:

Las capas de ggplot2

Cada componente del gráfico es una capa que se añade con +. Las capas se evalúan en orden y se apilan.

📊
Datos (data)
El data.frame con la información a representar
ggplot(data)
🎯
Aesthetics (aes)
Mapeo: qué variable va en x, y, color, tamaño…
aes(x, y, ...)
⬛
Geometrías (geom_*)
Forma visual: puntos, barras, líneas, cajas…
geom_point()
📏
Escalas (scale_*)
Controlan cómo los valores se mapean al espacio visual
scale_color_*
🔲
Facetas (facet_*)
División en subgráficos por variable categórica
facet_wrap(~var)
🎨
Tema (theme_*)
Estética global: fondo, grilla, tipografía
theme_minimal()

Sección 3

Aesthetics
Mapeando variables a propiedades visuales

La función aes() conecta columnas del dataset con propiedades visuales del gráfico. Es el corazón de ggplot2.

● Nivel: Inicial

Propiedades estéticas disponibles

Mapeo vs. configuración fija

¡Ojo! Hay una diferencia crucial entre mapear una aesthetic a una variable y fijar un valor constante:

R · mapeo_vs_fijo.R 
# ✅ MAPEO: color varía según la variable Species
ggplot(iris, aes(x = Petal.Length, y = Petal.Width,
                   color = Species)) +   # dentro de aes()
  geom_point()

# ✅ FIJO: todos los puntos en azul, siempre
ggplot(iris, aes(x = Petal.Length, y = Petal.Width)) +
  geom_point(color = "steelblue")     # fuera de aes()

# ❌ ERROR COMÚN: color literal dentro de aes() → hace leyenda
ggplot(iris, aes(x = Petal.Length, y = Petal.Width,
                   color = "steelblue")) +  # ← mal!
  geom_point()
⚠ Warning en tercer ejemplo:
  'color = "steelblue"' dentro de aes() crea una variable llamada
  "steelblue" y genera una leyenda de un solo valor. No es error
  de R pero sí un error conceptual frecuente.
⚠ Error frecuente

Poner colores o tamaños dentro de aes() cuando quieres un valor fijo es el error #1 de los principiantes en ggplot2. Recuerda: variable → dentro de aes(), constante → fuera de aes().

Barras e Histogramas

Para variables categóricas y distribuciones de frecuencia. Los más usados en reportes y dashboards.

● Nivel: Inicial

geom_bar vs geom_col

FunciónCuándo usarlaQué hace con y=
geom_bar() Tienes los datos crudos; quieres contar Cuenta las filas automáticamente (stat="count")

💡 Número de bins

El número de barras del histograma cambia drásticamente el interpretación. Prueba varias opciones: el valor por defecto (bins=30) no siempre es el mejor. También puedes usar binwidth para especificar el ancho en unidades de la variable.

# Pivotear a formato largo

  select(date, psavert, uempmed) |>

ggplot(economics_long, aes(x = date, y = valor, color = variable)) +

📌 Formato largo vs ancho

ggplot2 trabaja mejor con datos en formato largo (tidy): una columna por variable de respuesta, una fila por observación. La función pivot_longer() de tidyr transforma datos anchos a largos. Es fundamental para graficar múltiples series.

  geom_violin(alpha = 0.7, trim = FALSE) +
  geom_boxplot(width = 0.1, fill = "white", alpha = 0.8) +

💡 ¿Boxplot o violín?

Usa boxplot cuando la audiencia no es técnica y quieres destacar mediana y outliers. Usa violín cuando la forma de la distribución importa (bimodalidad, asimetría). Combinarlos es a menudo la mejor opción.

Sección 8

Escalas y Colores

Los colores no son decoración: son datos. Un buen uso del color mejora la comprensión; un mal uso la destruye.

● Nivel: Intermedio

Tipos de escalas de color

🎨

Cualitativas

Para categorías sin orden. Ej: Set1, Set2, Paired, Dark2. Cada color es diferente pero sin jerarquía.

📊

Secuenciales

Para valores numéricos de 0 a N. Ej: Blues, Greens, YlOrRd. El tono indica magnitud.

↔️

Divergentes

Para valores con un punto central (ej: 0). Ej: RdBu, PiYG. Los extremos tienen colores opuestos.

♿

Accesibles

Okabe-Ito, viridis. Diseñadas para ser distinguibles por personas con daltonismo.

R · escalas_color.R 
# ColorBrewer para categorías
scale_color_brewer(palette = "Set1")
scale_fill_brewer(palette = "Paired")

# Viridis: perceptualmente uniforme y daltónico-friendly
scale_color_viridis_c()              # continua
scale_color_viridis_d()              # discreta

# Colores manuales
scale_color_manual(
  values = c("A" = "#e74c3c", "B" = "#3498db", "C" = "#2ecc71")
)

# Gradiente personalizado
scale_color_gradient(low = "#ffffcc", high = "#800026")
scale_color_gradient2(
  low = "#2980b9", mid = "white", high = "#c0392b", midpoint = 0
)

⚠ Nunca uses colores del arcoíris

El mapa de colores "jet" (arcoíris) distorsiona la percepción, no es accesible para daltónicos y crea artefactos visuales. Usa siempre viridis, plasma u otra escala perceptualmente uniforme para datos continuos.

Sección 9

Facetas — Small Multiples

Las facetas dividen un gráfico en múltiples paneles según una variable categórica. Permiten comparar subgrupos con el mismo eje.

● Nivel: Intermedio

facet_wrap — distribución en cuadrícula

R · facet_wrap.R 
ggplot(mpg, aes(x = displ, y = hwy, color = drv)) +
  geom_point(alpha = 0.7) +
  geom_smooth(method = "lm", se = FALSE) +
  facet_wrap(
    ~class,          # dividir por clase de vehículo
    ncol = 3,        # 3 columnas
    scales = "free"  # ejes independientes por panel
  ) +
  scale_color_brewer(palette = "Set1") +
  theme_minimal(base_size = 11)
[Plot] 7 paneles (uno por clase) · scales="free" permite
  ejes independientes en cada panel para mejor uso del espacio.
  
⚠ Con scales="free" ya no puedes comparar magnitudes entre paneles.

facet_grid — cuadrícula bidimensional

R · facet_grid.R 
# Filas: drv (tracción), Columnas: cyl (cilindros)
ggplot(mpg, aes(x = displ, y = hwy)) +
  geom_point(color = "#3498db", alpha = 0.6) +
  facet_grid(drv ~ cyl) +   # filas ~ columnas
  theme_minimal(base_size = 10)
FunciónCuándo usarLimitación
facet_wrap(~x) Una variable, muchos niveles El orden puede no ser intuitivo
facet_grid(x ~ y) Dos variables, pocas categorías Se vuelve confuso con muchos niveles
scales = "free" Rangos muy distintos entre grupos No se puede comparar entre paneles
scales = "fixed" Quieres comparación directa Puede desperdiciar espacio
Sección 10

Temas y Estética Final

El tema controla todos los elementos no-datos del gráfico: fondo, grilla, tipografía, márgenes, leyendas. Un buen tema puede transformar un gráfico mediocre en uno de presentación.

● Nivel: Avanzado

Temas predefinidos

theme_gray()

Por defecto. Fondo gris claro, grilla blanca.

theme_minimal()

Limpio, sin fondo. Ideal para publicaciones.

theme_classic()

Estilo "paper científico". Solo ejes L.

theme_dark()

Fondo oscuro. Para dashboards y presentaciones.

theme_void()

Sin ejes ni fondo. Para mapas y diagramas.

theme_bw()

Blanco y negro. Ideal para impresión.

Personalización con theme()

R · tema_personalizado.R 
# Crear un tema corporativo reutilizable
mi_tema <- theme_minimal(base_size = 13) +
  theme(
    # Tipografía del título
    plot.title    = element_text(face = "bold", size = 16,
                                  margin = margin(b = 5)),
    plot.subtitle = element_text(color = "#666", size = 12),
    plot.caption  = element_text(color = "#999", size = 9,
                                  hjust = 0),
    
    # Leyenda
    legend.position  = "bottom",
    legend.title     = element_text(face = "bold", size = 11),
    
    # Grilla
    panel.grid.minor = element_blank(),
    panel.grid.major = element_line(color = "#eee"),
    
    # Fondo
    plot.background  = element_rect(fill = "white", color = NA),
    
    # Márgenes del gráfico completo
    plot.margin = margin(20, 20, 10, 20)
  )

# Usar el tema guardado
ggplot(iris, aes(Sepal.Length, Petal.Length, color = Species)) +
  geom_point() + mi_tema
